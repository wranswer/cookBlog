    欢迎大家投稿

    开放投稿的栏目：JS代码，PNG图标，设计分享，QQ表情

    <JS代码>投稿说明：
    1. 请使用JS代码模板 右键另存为
    2. 压缩包需要包含js，images css index.html index.jpg（缩略图，尺寸为280*140)
    3. 注明原作者和出处。
    4. 作为鼓励，我们可在代码预览页保留投稿者的网站或博客链接。

    <QQ表情>投稿说明：
    1. 内容新鲜有趣，健康向上。无网址链接。
    2. 所有图片压缩包。
    3. 相关文字介绍和eif格式表情包。
    4. 注明原作者和出处。
    QQ表情投稿请直接发送邮件至：511401068@qq.com

    <PNG图标>投稿说明：
    1. 尺寸大小统一，16x16 32x32 64x64 128x128 256x256 512x512，格式为.png
    2. 缩略图模板 右键另存为
    3. 图标文件命名只能为英文字母，数字和下划线，请勿使用中文命名。
    3. 注明原作者和出处。

    <设计分享>投稿说明：
    1. 内容新鲜有趣，健康向上。
    2. 缩略图尺寸240x160px
    3. 正文图片宽度须在600px以上，1000px以下
    4. 附相关文字，注明原作者和出处。

    以下原因可能会导致你的投稿不被通过审核：
    1. 内容违反国家相关规定。
    2. 跟本站栏目无关。
    3. 明显不符合事实，个人炒作。
    4. 企业软文，但投稿者非本站企业用户。
    5. 文章可读性不高。
    6. 抄袭别人的稿子。 

    投稿应该注意些什么？
    1、原作文章，请在标题处注明“原创”字样；
    2、对于非原作文章，请尊重原作者的权益，请注明文章的作者、出处、出处链接。如果不提供这些信息，文章将不被通过验证。
    3、文章和新闻中不能含有外部链接和超级链接；
    4、本站接受文章/新闻的前提是该文对广大用户有所帮助，纯粹的广告文字将不予通过审核。
    5、与站长相关或者与互联网密切相关。
    6、标题力求简短、醒目、新颖、吸引人，杜绝标题党，标题尽量不少于9个汉字，不多于22个汉字。
    7、文章正文内的图片尺寸不得超过规范，即：600px以上，1000px以下 。
    8、文章新闻应避免用 "今天"、"昨天"，应改成具体日期。
    9、统一用中文标点符号，英文的不要用，除非是英文或者数字间是可以用，标题不要用叹号和逗号、问号、顿号、破折，括号可以用；还有文章中禁止表情符号。
    10、 由于投稿数量众多，删稿不再通知，请作者发布文章前先自留稿底。
